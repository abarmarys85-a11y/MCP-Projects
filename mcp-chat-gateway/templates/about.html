About This Chat Interface

How It Works

This is a chat that answers questions exclusively using available MCP tools.

What to Expect

  • The AI must call at least one tool for every question you ask
  • The AI never answers from its own knowledge — only uses tool results
  • If no tool can answer your question, you will see this response: "We don't have any internal tool to answer your question."

How to Use

Type your question in the input field at the bottom. The AI will automatically determine which tools to call based on your question and provide answers based solely on the tool results.

Technical Details

This chat gateway is built with Flask (Python) and connects to an AI provider (DeepSeek, OpenAI, Ollama, or any OpenAI-compatible API). The AI receives tool definitions from an MCP server and uses them to answer your questions.

For more information, visit the OKFN MCP CKAN server repository.
